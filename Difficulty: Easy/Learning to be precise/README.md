<h2><a href="https://www.geeksforgeeks.org/problems/learning-to-be-precise/1?page=1&category=Java&sortBy=submissions">Learning to be precise</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">There are times when your answer is a floating point that contains undesired amount of digits after decimal. </span></p>
<p><span style="font-size: 18px;">Here, we'll learn<strong> </strong>how to get a precise answer out of a floating number. You are given two floating numbers <strong>a</strong> and <strong>b</strong>. </span></p>
<p><span style="font-size: 18px;">You need to print the value of <strong>a/b</strong> rounded to <strong>exactly 3</strong> decimal places.</span></p>
<ul>
<li><span style="font-size: 18px;">You may use library precision methods.</span></li>
<li><span style="font-size: 18px;">You need to print the result (return type is void) without any newline or space around it.</span></li>
</ul>
<p><span style="font-size: 18px;"><strong>Example:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>a = 5.43, b = 2.653
<strong>Output: </strong>2.047
<strong>Explanation: </strong>The exact result of a / b is approximately 2.04674. When rounded to 3 decimal places, it becomes 2.047.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>a = 10.0, b = 3.0<strong>
Output: </strong>3.333<strong>
Explanation: </strong>10 / 3 = 3.33333... → Rounded to 3 decimal places = 3.333</span></pre>
